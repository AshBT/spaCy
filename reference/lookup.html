Lexical Lookup¶

Where possible, spaCy computes information over lexical types, rather than tokens. If you process a large batch of text, the number of unique types you will see will grow exponentially slower than the number of tokens — so it’s much more efficient to compute over types. And, in small samples, we generally want to know about the distribution of a word in the language at large — which again, is type-based information.

You can access the lexical features via the Token object, but you can also look them up in the vocabulary directly:

>>> from spacy.en import English
>>> nlp = English()
>>> lexeme = nlp.vocab[u'Amazon']
