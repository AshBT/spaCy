spacy.en.EN¶

Tokenizer API¶

Lexeme Features Flag IDs¶

A number of boolean features are computed for English Lexemes. To access a feature, pass its ID to the spacy.word.Lexeme.check_flag() function.

Orthographic Features¶

These features describe the orthographic (lettering) type of the word. The function used to compute the value is listed along with the flag.

Distributional Orthographic Features¶

These features describe how often the lower-cased form of the word appears in various case-styles in a large sample of English text. See spacy.orth.oft_case()

Tag Dictionary Features¶

These features describe whether the word commonly occurs with a given part-of-speech, in a large text corpus, using a part-of-speech tagger designed to reduce the tag-dictionary bias of its training corpus. See spacy.orth.can_tag().
